## Graphic Design Principles

### Stylization

**Approach:** Very light. Reality first, design second. Graphics should feel placed, not designed.

**Action items:**
- No drop shadows on text (or very subtle if absolutely needed)
- No black strokes
- No deep glows
- No heavy gradients
- If a graphic looks "designed," simplify it until it looks "placed"

### Animation Quantity

**Approach:** Minimal (A-roll heavy). Most of the video should be talking head. Only add graphics when there's a specific reason.

**Action items:**
- Add graphics to illustrate concepts, show data, or visualize frameworks
- Don't add motion just to fill space
- If you're adding an animation just because "it's been a while," don't

### Motion Per Animation

**Approach:** Subtle movement. Motion should underline ideas, not decorate.

**Action items:**
- No bounce expressions
- No overshoot
- No spring physics
- Scale animations: 100% to 102-104% max
- Position animations: 10-20px max movement
- Fade animations: 8-12 frames
- If motion feels noticeable, it's too much

### Motion Style

**Approach:** Smooth-bezier. Elegant, flowing easing curves.

**Action items:**
- Use ease-in-out on all keyframes
- No linear keyframes
- No bounce or overshoot expressions
- Movement should feel inevitable, not energetic

### Pacing Baseline

**Approach:** Breathing room. Intentional silences, not rushed, lets moments land.

**Action items:**
- Let shots breathe
- Hold frames slightly longer than feels natural
- Silence is allowed
- Don't cut just because nothing is happening â€” cut when something new begins
- Don't add music hits or sound effects to fill silence

## The Visual Litmus Test

Before finalizing any visual:

1. **Does it feel like a luxury brand ad?** If it feels cheap or loud, simplify.
2. **Could this be a frame from an A24 film?** If it feels like a YouTube thumbnail, pull back.
3. **Is every element intentional?** If something is decorative without purpose, remove it.
4. **Does it breathe?** If it feels cluttered or busy, reduce.
